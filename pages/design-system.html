Design System

מערכת עיצוב

The unified visual language for Odysee — Personal AI Travel Concierge. Dark, refined, terracotta-accented. All components use CSS custom properties and scale between light, dark, and system themes.

שפת העיצוב האחידה של Odysee — יועץ הנסיעות האישי המבוסס AI. כהה, מעודן, עם הדגשת טרה-קוטה. כל הרכיבים משתמשים ב-CSS custom properties ומתאימים את עצמם לערכות Light, Dark ו-System.
